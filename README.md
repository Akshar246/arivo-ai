# Arivo AI

An AI-powered career platform for international students
navigating the UK job market.

## The Problem

International students face unique challenges in the UK —
visa sponsorship filters, cultural differences, and zero
local network. No existing platform addresses this.

## Solution

Arivo AI provides visa-intelligent job matching, AI-powered
skill gap analysis, and an interview coach trained on
UK-specific career data.

## Tech Stack

- **Frontend:** React, Chart.js
- **Backend:** Node.js, Express, MongoDB
- **AI Service:** Python, FastAPI, LangChain, ChromaDB
- **LLM:** Groq API (LLaMA 3) / Ollama (local)
- **Embeddings:** HuggingFace all-MiniLM-L6-v2
- **Deployment:** AWS EC2, Docker, GitHub Actions
